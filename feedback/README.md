# 📦 APM Feedback

Thanks for trying [APM](https://github.com/microsoft/apm)! Drop your feedback here to earn **+10 bonus points**.

## How to Submit

**Option A — PR to this folder:**

1. Create a markdown file named `<team-name>.md` (e.g., `acme-agents.md`)
2. Include:
   - **Team name**
   - **What you used APM for** (managing skills, plugins, MCP servers, etc.)
   - **What worked well**
   - **What could be improved**
3. Open a PR to this repo

**Option B — Issue on the APM repo:**

1. Open an issue at [microsoft/apm/issues](https://github.com/microsoft/apm/issues)
2. Include the same info above + a screenshot or link to your `apm.yml` as proof
3. Reference your hackathon submission in the issue

> Either option counts — pick whichever is easier for your team!
